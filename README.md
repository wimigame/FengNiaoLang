# 蜂鸟编程语言
基于LLVM的编程语言，支持中文重命名方法

2023-01-27 运行今天的命令将得到这样一个结果
```bash
    $ mkdir build
    $ cd build
    $ cmake .. && make
    $ ./fengniao
    $ 编程语言从这里就开了！哈哈
```
